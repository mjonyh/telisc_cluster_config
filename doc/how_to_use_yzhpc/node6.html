Useful Linux Command

Use man COMMAND to do more with the following commands:

ls
Very useful commands for documents listing in the current directory.
cd
To jump into another directory.
mkdir
Use it for creating new directory.
rm
To delete document it is used.
mv
To rename document, it is used.
cp
To copy document, it is used.
man
To view the manual of a package, it is used.
tail
Very useful command for viewing live output.
head
It shows the top few lines of a document.
less
It shows the document in visual mode.
cat
Shows all the content of a document.
