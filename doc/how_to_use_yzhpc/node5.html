From Windows Computer

A number of ssh client is available for windows machine. PuTTY is one of the best ssh client for windows machine.

  1. Open PuTTY from start menu.
  2. Provide username and address (i.e. yzhpc.sust.edu).
  3. Login to the system.
  4. Execute exit command to logout.
